# 发布 MooM 空闲时间表

这个文件夹是独立网站，不需要和 `learning-records` 一起上传。

## 上传文件

只需要上传：

- `index.html`
- `style.css`
- `README.md`
- `.nojekyll`

## GitHub Pages

1. 新建一个 GitHub 仓库，例如 `moom-schedule`。
2. 上传这个文件夹里的文件。
3. 打开仓库的 `Settings`。
4. 进入 `Pages`。
5. Source 选择 `Deploy from a branch`。
6. Branch 选择 `main`，Folder 选择 `/ (root)`。
7. 保存后等待 1 到 3 分钟。

生成的网址类似：

```text
https://你的用户名.github.io/moom-schedule/
```

这个网址就是可以扫码或点击打开的公网链接。
