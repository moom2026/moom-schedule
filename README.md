# MooM的空闲时间表

这是一个独立的静态网页，和找工作的个人主页分开使用。

页面内容：

- 标题：`MooM的空闲时间表`
- 范围：从访问当天开始的三个月
- 状态：目前每天、每个时间段都显示 `空闲`
- 预约申请：点击空闲时间后填写表单，可生成预约信息
- 文件：`index.html`、`style.css`

## 本地预览

在这个文件夹运行：

```bash
python3 -m http.server 8020
```

然后打开：

```text
http://localhost:8020
```

## 修改预约时间

预约时间在 `index.html` 里的 `timeSlots` 修改：

```js
const timeSlots = ["09:00", "10:00", "11:00", "13:00", "14:00", "15:00", "16:00", "17:00"];
```

## 预约通知邮箱

预约表单会先在页面上生成预约信息，并提供两个操作：

- 复制预约信息
- 打开 Gmail 网页版，自动填好收件人 `shunyoutou@gmail.com`、标题和内容

预约人需要在打开的 Gmail 页面中点击发送。纯静态网页不能自动替别人发送邮件。

要做到真正自动发送到 LINE、微信、Telegram、Discord 或邮箱，需要再接一个后端、表单服务或平台机器人。
